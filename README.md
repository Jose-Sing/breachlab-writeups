## Writeup Philosophy

Each writeup documents:

- The **thinking process** — hypotheses, dead ends, pivots
- The **technique** — what it is, why it works
- The **takeaway** — what this maps to in a real environment

What's never published: passwords, flags, or anything that skips the learning for the next operator.
